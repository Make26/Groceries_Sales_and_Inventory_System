# Inventory and Sales Management System

This is a PHP-based Inventory and Sales Management System using MySQL and Bootstrap.

## Features
- Product listing and inventory tracking
- Sales processing with receipt printing
- Payment modal with change calculation
- User login and sales tracking

## Setup
1. Import `sql/database.sql` into your MySQL server
2. Update `db_connect.php` with your database credentials
3. Run `index.php` via localhost
